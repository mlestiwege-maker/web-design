# Mufutumari Shop Web Design

A modern multi-page ecommerce website built with **HTML, CSS, Bootstrap 5, and vanilla JavaScript**.

This project keeps its original structure while adding professional marketplace-style features inspired by large ecommerce platforms (trust cues, flash deals, interactive shop tools, cart flow, order tracking, and consistent responsive UI).

## Live Project Structure

- `My First Webpage/FirstPage.html` — Home
- `My First Webpage/SecondPage.html` — Shop
- `My First Webpage/ThirdPage.html` — About
- `My First Webpage/FourthPage.html` — Blog
- `My First Webpage/FifthPage.html` — Contact
- `My First Webpage/SixthPage.html` — Services
- `My First Webpage/SeventhPage.html` — FAQ
- `My First Webpage/EighthPage.html` — Checkout
- `My First Webpage/NinthPage.html` — Orders
- `My First Webpage/TenthPage.html` — Track Order
- `My First Webpage/mystyle.css` — Global styles and theme system

## Key Features

### Core Website
- Multi-page responsive navigation
- Professional homepage sections (hero, featured products, testimonials, lookbook, newsletter)
- Consistent visual design and image sizing across sections/pages
- Footer with contact/social/payment details

### Marketplace-Style Enhancements
- Trust strip and category cards on homepage
- Flash deals section with live countdown
- Supplier/buyer value section (verified quality, bulk-friendly offers, regional delivery messaging)
- Shop trust badges and trending search chips
- Recently viewed products in shop
- Contact page upgraded to a buyer support + RFQ hub with inquiry type, quantity, budget, and preferred contact time
- Premium product cards with quick-view and wishlist actions
- One-tap buyer contact actions (Call, Email, WhatsApp) on key pages

### Shopping Experience
- Product search + category filter + sorting
- Cart drawer with quantity controls and remove/clear actions
- Shipping progress bar toward free-shipping target
- Checkout workflow with coupon support
- Order history and order tracking pages
- Contact inquiry capture and quick FAQ support section

### UX & UI
- Dark mode (saved with `localStorage`)
- Page loader animation
- Back-to-top button
- Hover micro-interactions and polished card styling
- Floating quick-contact widget for faster buyer communication

## Browser Technologies

- HTML5
- CSS3
- Bootstrap 5.3.3
- Font Awesome 5
- Vanilla JavaScript (no framework)
- `localStorage` for lightweight persistence

## LocalStorage Keys Used

- `mufu-theme` — dark/light theme preference
- `mufu-wishlist` — wishlist product IDs
- `mufu-cart` — cart items and quantities
- `mufu-orders` — placed orders
- `mufu-recently-viewed` — recently viewed shop products
- `mufu-contact-leads` — recent contact/RFQ submissions

## Run Locally

Because this is a static website, you can run it in either of these ways:

1. Open `My First Webpage/FirstPage.html` directly in your browser.
2. Or run a local static server and open the served URL.

## Design Goal

The goal is to keep the original project identity and progressively enhance it with selected high-impact ecommerce features (without completely rebuilding the project from scratch).
